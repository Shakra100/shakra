Selected Projects

Here are some of my selected projects which I did either for academic purposes or for fun:

Breast Cancer Detection and Segmentation using Convolutional Neural Networks

• Tumor Detection using Mobilenet
Approach:
I evaluated several CNN models including VGG16, ResNet50, and MobileNet to detect tumors from mammograms and histopethalogical images. For training and testing CBIS-DDSM dataset is used which is an updated and standardized version of the Digital Database for Screening Mammography (DDSM).
Outcome:
Trained MobileNet models which are able to accurately classify the tumor into benign and malignant classes
• Tumor Segmentation using UNet
Approach:
I used the UNet model for training on a publically available dataset namely Data Science Bowl 2018. Training dataset contains images along with the mask of nuclei present in images. During training, technique of augmentation is used to reduce overfitting.
Outcome:
A trained UNet model is able to segment the nuclei present in images.

Auto diabetes detection using logistic regression on the microbiome of human body

Approach:
The approach used for this project was training the logistic regression model on microbiome data for classification of diabetic and normal class.
Outcome:
A trained model was able to predict diabetic or normal class on the basis of microbiome data given as input.

Identification and Extraction of specific human proteins using dictionaries and suffix trees

Approach:
The technique for this project was using suffix tree and dictionary to store data and find the desired search. The project was built in Python language.
Outcome:
User can find specific human protein along with protein's detail from alot of human proteins by narrowing down his search from dictionary.

Restaurant Management System with online reservation system using Object Oriented Programming

Approach:
The system was built in JAVA language using Object Oriented Programming techniques like inheritance, Abstraction, and Polymorphism.
Outcome:
Restaurant Management system is used by manager, and staff for managing the hotel and for online reservation by customers.

Wampus World game

Approach:
The game was built in Python language using dictionary.
Outcome:
In Wampus World game, there were 16 blocks having GOLD, PITS and Smell present in random blocks. The agent has to move with different arrows and to get the GOLD for winning the game. If he reaches the block with PIT or Smell, he failed. The game code was built in Python.
